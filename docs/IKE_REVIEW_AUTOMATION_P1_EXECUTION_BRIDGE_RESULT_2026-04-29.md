## Summary

Implemented a bounded execution bridge for the L1 review runner.

- Added explicit `--execute none|qoder|openclaw`.
- Kept default behavior as prepare-only with `--execute none`.
- Preserved `--dry-run` as no-write and no-delegate execution.
- Added delegate invocation through existing Qoder and OpenClaw wrappers after artifact preparation.
- Captured delegate exit code and returned non-zero when a delegate wrapper fails.
- Added `--delegate-timeout` with a conservative default of 1800 seconds for delegated wrapper execution.
- Extended done JSON only with minimal attempt/status fields.

## Why This Solution

- Keeps the runner conservative: it prepares deterministic artifacts first, then optionally launches exactly one selected wrapper.
- Reuses existing `scripts/qoder/qoder_delegate.py` and `scripts/acpx/openclaw_delegate.py` interfaces instead of adding new orchestration machinery.
- Keeps `--dry-run` safe by returning before any directory creation, file write, or delegate process launch.
- Bounds delegated execution with `subprocess.run(..., timeout=...)` so a hung wrapper fails closed instead of hanging the runner.
- Records only execution attempt metadata and does not parse reviewer judgment, promote work, or make reviewer output authoritative.
