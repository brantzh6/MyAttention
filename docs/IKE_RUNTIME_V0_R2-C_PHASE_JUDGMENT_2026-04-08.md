## Why This Phase Exists

- a real task lifecycle

- one controller-facing or settings-facing read surface only
- no broad knowledge-base redesign
- no generic object API expansion
- no notification mesh
- no graph-memory layer

Open `R2-C` as the next active runtime phase.
