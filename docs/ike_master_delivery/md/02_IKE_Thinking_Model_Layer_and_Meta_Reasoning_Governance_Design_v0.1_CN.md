# IKE 科学思考模型层与元推理治理设计稿 v0.1

## 1. 为什么需要思考模型层

如果没有显式的思考模型层，知识大脑只能整理信息，进化大脑只能对新鲜事物做反应。真正的科学理解需要明确的推理模式和领域范式。

## 2. 初始策略

采用 A→D 的演化路径。A：固定核心模型集合；B：固定核心 + 领域适配器；C：通用推理层 + 领域范式层双层结构；D：模型 portfolio / 模型市场式治理。D 是长期目标。

## 3. 7+N 初始方案

初始核心集合覆盖解释生成、信念更新、反驳机制、因果分析、系统思维、范式跟踪和研究纲领评估；再叠加领域范式包形成可工作的起步版本。

## 4. 元推理治理层

元推理治理层负责选择模型、组合模型、评估结果，并持续更新模型库。本质上它管理的是“系统如何思考”。

## 5. 初始化与进化

初始化应由人工策划的最小模型集和领域 starter pack 启动；后续进化依赖任务匹配、证据形态、历史表现和 harness 结果来驱动。
