## Key Design Decisions

1. **Transitional API stance**: preview/inspect style endpoints, not durable GET
2. **Provisional/experimental stability markers**: All objects marked as provisional
3. **No fake durability promises**: Chain inspect reads from TaskArtifact, doesn't promise stable retrieval
4. **TaskArtifact as honest substrate**: Chain persistence via existing TaskArtifact.extra
5. **Additive helpers only**: Each packet adds new helpers, doesn't rewrite existing ones
6. **Explicit traceability**: All helpers preserve source object IDs in references and provenance
