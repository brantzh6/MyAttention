# IKE 领域范式库与研究卡规范 v0.1

## 1. 为什么需要范式库

不同领域需要不同研究范式。一个领域不能只被表示为实体和事实，也必须包含对该领域进行解释和研究的竞争性方式。

## 2. 核心对象

将 Paradigm 和 ResearchCard 作为一等对象。Paradigm 描述核心假设、偏好的证据类型、典型问题和常见失效模式；ResearchCard 则把具体研究任务绑定到一个或多个范式上。

## 3. Starter Pack

应先为 AI、经济学、生命科学等优先领域准备初始范式包。

## 4. 进化

范式可以被吸收、拆分、合并、降级、冻结或退休，范式库本质上是动态演化的。
