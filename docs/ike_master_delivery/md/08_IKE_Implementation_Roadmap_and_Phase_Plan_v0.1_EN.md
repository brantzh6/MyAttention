# IKE Implementation Roadmap and Phase Plan v0.1

## 1. Roadmap Principle

Use A→D evolution explicitly. A/B are the practical starting modes. D is the long-term target and should be written down to avoid accidental local optimization.

## 2. Phase Plan

M0: principle and bootstrap. M1: object model + workflows + initial harness. M2: model governance and paradigm packs. M3: experimentation and decision loop. M4: portfolio-style model governance.

## 3. Priority Streams

Priority streams are: object/schema stream, workflow/task engine stream, and harness/governance stream.

## 4. Avoid Drift

Do not let the system devolve into a crawler + vector DB + chat wrapper. The long-term target is a self-improving research system.
