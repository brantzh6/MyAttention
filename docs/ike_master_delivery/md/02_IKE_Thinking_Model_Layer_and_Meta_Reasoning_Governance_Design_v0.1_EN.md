# IKE Thinking-Model Layer and Meta-Reasoning Governance Design v0.1

## 1. Why a Thinking-Model Layer

Without an explicit thinking-model layer, the Knowledge Brain can only rearrange information, and the Evolution Brain can only react to novelty. Scientific understanding requires explicit reasoning patterns and field paradigms.

## 2. Initial Strategy

Use an A→D evolution path. A: fixed core model set. B: fixed core + field adapters. C: dual-layer general reasoning + field paradigms. D: model portfolio / market-style governance. D is the long-term target.

## 3. Initial 7+N Approach

The suggested initial core set covers explanation generation, belief updating, refutation, causality, systems thinking, paradigm tracking, and research-program evaluation. Field-specific paradigm packs extend this base.

## 4. Meta-Reasoning Governance

A meta-reasoning governance layer selects models, combines models, evaluates outcomes, and updates the model library over time. It manages how the system thinks.

## 5. Initialization and Evolution

Initialization should be bootstrapped by a human-curated minimum model set and domain starter packs. Evolution should be driven by task fit, evidence shape, historical performance, and harness outcomes.
