## 五大核心模块

### 1. 信息流 (Information Feed)

**职责**：信息源管理、获取、处理、展示、导入知识库

| 功能 | 说明 |
|------|------|
| 源管理 | RSS 订阅、URL 监控、API 接入 |
| 权威分级 | S/A/B/C 四级分类（政府、厂商、媒体、社区） |
| 深度获取 | L1 摘要 → L2 全文 → L3 深度分析 |
| 反爬处理 | 自动检测、切换代理、云服务 |
| 导入知识库 | 一键转入知识库 |
| 分类筛选 | 按分类、重要性、来源筛选 |

---

### 2. 智能对话 (Intelligent Chat)

**职责**：多模型交互、搜索、查询、研究、决策投票

| 功能 | 说明 |
|------|------|
| 多模型支持 | Qwen、GLM、Kimi、Claude、GPT |
| RAG 搜索 | 知识库向量检索 |
| 多模型投票 | 多模型同时回答，取最佳 |
| 深度研究 | 多次对话、多源交叉验证 |
| 实时搜索 | 联网搜索最新信息 |
| 决策投票 | 多模型投票决策，置信度评估 |

---

### 3. 知识管理 (Knowledge Management)

**职责**：知识库存储、向量化、图谱组织

| 功能 | 说明 |
|------|------|
| 知识库 | 多知识库支持 |
| 向量存储 | Qdrant 向量数据库 |
| 知识图谱 | 实体、关系、推理 |
| 分类体系 | 层次化知识分类 |
| 知识链接 | 信息 → 知识关联 |

---

### 4. 知识大脑 (Knowledge Brain)

**职责**：深度学习各学科知识，建立世界运行的知识体系

| 功能 | 说明 |
|------|------|
| 实体提取 | 从文本识别实体（人物、组织、概念、技术） |
| 关系抽取 | 建立实体间关联（因果、包含、相关） |
| 知识推理 | 基于已有知识推断新关系 |
| 跨学科关联 | 建立跨领域知识连接 |
| 世界模型 | 理解世界运行规律 |

---

### 5. 进化大脑 (Evolution Brain)

**职责**：自我测试、感知变化、持续优化

| 功能 | 说明 |
|------|------|
| 自动测试 | API 测试 + UI 自动化测试 |
| 问题发现 | 自动识别 Bug、性能问题、UI 问题 |
| 性能监控 | 加载时间、响应速度、资源使用 |
| AI 决策 | 外部 Agent + 本地 LLM + 规则 |
| 反爬处理 | 自动检测、切换策略 |
| 策略优化 | 自动调整源配置、获取深度 |

## 数据流

```
信息源 → 信息流 → 知识库 + 知识图谱 → 智能对话 → 决策支持
                ↑
                │
           进化大脑（持续优化）
                │
    测试/监控/AI决策 → 系统自我进化
```
