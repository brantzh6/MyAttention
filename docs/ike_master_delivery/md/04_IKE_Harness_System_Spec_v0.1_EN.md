# IKE Harness System Spec v0.1

## 1. Harness Positioning

In IKE, harness is not just testing infrastructure. It is self-verification infrastructure for sensing quality, knowledge quality, reasoning quality, and evolution quality.

## 2. Five Harness Layers

Information harness, Knowledge harness, Evolution harness, Thinking-model harness, and Governance harness.

## 3. Story-Level Harness

Priority stories include: field foundation building, entity-centered research, problem-driven judgment, new concept detection and adoption, paradigm comparison, and system self-upgrade evaluation.

## 4. Scoring Principles

Use deterministic checks, structured semantic checks, and model-judged checks only where necessary. Every evaluation artifact should be traceable.

## 5. Cadence

Use lightweight pre-merge harness, broader nightly harness, and monthly governance review harness.
