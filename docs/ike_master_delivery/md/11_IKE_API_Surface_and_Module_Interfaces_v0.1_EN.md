# IKE API Surface and Module Interfaces v0.1

## 1. External API Families

Recommended families: observations, knowledge, research, governance, harness, and administration.

## 2. Internal Interfaces

Modules should communicate through application-layer contracts and events, not through direct database coupling.

## 3. Response Standards

Responses should carry provenance, confidence, and traceability hooks.

## 4. Phase-1 API Set

Phase-1 should prioritize workflow-triggering endpoints and object CRUD/query endpoints needed by bootstrap tasks.
