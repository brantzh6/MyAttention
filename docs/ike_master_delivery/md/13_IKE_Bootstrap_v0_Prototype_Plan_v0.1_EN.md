# IKE Bootstrap v0 Prototype Plan v0.1

## 1. What v0 Must Prove

v0 must prove that IKE can ingest targeted observations, build basic structured knowledge, select an initial reasoning/paradigm configuration, run a research workflow, and close at least one task/experiment/decision loop.

## 2. What v0 Must Not Do

Do not overbuild orchestration, distributed services, or a giant generic agent framework.

## 3. Starter Pack

Use a small set of domains, objects, APIs, workflows, and harness cases.

## 4. Exit Criteria

v0 exits when the bootstrap loop is real, inspectable, and repeatable.
