# IKE Source Intelligence V1 - M7 Phase Judgment

Date: 2026-04-14
Phase: `M7 AI-Assisted Candidate Judgment Inspect`
Status: `activated`

## Why This Slice

The project has already recorded that recent heuristic-heavy slices are only
cleanup scaffolds, not the end-state architecture.

The next bounded step should therefore reopen AI participation above the
cleaned discovery surface without widening into persistence, auto-promotion,
or a general agent-control system.

## Scope

Add one inspect-only AI judgment lane that:

1. reuses the existing `/sources/discover` path
2. judges only a bounded subset of returned candidates
3. returns advisory verdicts only
4. does not persist results
5. does not change source-plan truth

## Out Of Scope

This slice does not attempt:

1. model voting or panel judgment
2. persistence into source plans or canonical truth
3. automatic follow/subscribe decisions
4. dynamic methodology-program generation
5. replacement of the existing controller gate

## Expected Output

One new inspect route that demonstrates a first closed-loop shape in which:

- discovery is still bounded and auditable
- AI participates in candidate-value judgment
- the output remains reviewable and non-authoritative
