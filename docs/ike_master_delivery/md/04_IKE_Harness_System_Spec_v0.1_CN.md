# IKE Harness 体系规范 v0.1

## 1. Harness 的定位

在 IKE 中，harness 不只是测试工具，而是对感知质量、知识质量、思考质量和进化质量进行自我验证的基础设施。

## 2. 五层 Harness

包括 Information harness、Knowledge harness、Evolution harness、Thinking Model harness 和 Governance harness。

## 3. 故事级 Harness

高优先故事包括：领域底座建模、实体中心研究、问题导向判断、新概念发现与引入、范式比较，以及系统自升级评估。

## 4. 评分原则

结合确定性检查、结构化语义检查，以及必要时的模型判分。所有评估工件必须可追溯。

## 5. 运行节奏

建议建立轻量 pre-merge harness、夜间回归 harness 和月度治理 review harness。
